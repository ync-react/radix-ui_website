## Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-aspect-ratio
```

## Anatomy

Import the component.

```jsx
import * as AspectRatio from '@radix-ui/react-aspect-ratio';

export default () => <AspectRatio.Root />;
```
