## Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-switch
```

## Anatomy

Import all parts and piece them together.

```jsx
import * as Switch from '@radix-ui/react-switch';

export default () => (
  <Switch.Root>
    <Switch.Thumb />
  </Switch.Root>
);
```
