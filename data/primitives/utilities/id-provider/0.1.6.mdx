## Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-id
```

## Anatomy

Import the component.

```jsx
import { IdProvider } from '@radix-ui/react-id';

export default () => <IdProvider />;
```

## API Reference

### IdProvider

When server rendering your app, you need to wrap it with the `IdProvider` component to ensure stable ids between server and client. You can learn more about this in our [server side rendering guide](/docs/primitives/overview/server-side-rendering).
